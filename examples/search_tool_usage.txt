============================================================================
SWITCHING TO HYBRID MODE (If You Prefer)
============================================================================

If you prefer to see all tools immediately without searching:

1. Edit claude_desktop_config.json
2. Change: "TOOLS_MODE": "search_only"
   To:     "TOOLS_MODE": "hybrid"
3. Restart Claude Desktop

Then all 7 tools will be visible immediately!
